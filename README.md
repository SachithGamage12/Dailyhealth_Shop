## ğŸ”§ Installation & Setup

<details>
<summary><b>ğŸ“¥ Click to view installation instructions</b></summary>

### Prerequisites
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- Composer (optional)

### Step-by-Step Guide

```bash
# 1. Clone the repository
git clone https://github.com/SachithGamage12/dailyhealth_shop.git

# 2. Navigate to project directory
cd dailyhealth_shop

# 3. Create database
mysql -u root -p
CREATE DATABASE dailyhealth_db;
exit;

# 4. Import database schema
mysql -u root -p dailyhealth_db < database/schema.sql

# 5. Configure database connection
cp includes/db_connection.example.php includes/db_connection.php
# Edit db_connection.php with your credentials

# 6. Set permissions
chmod -R 755 assets/
chmod -R 777 uploads/

# 7. Start development server
php -S localhost:8000
```

### Configuration

Edit `includes/db_connection.php`:

```php
<?php
define('DB_HOST', 'localhost');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
define('DB_NAME', 'dailyhealth_db');
?>
```
